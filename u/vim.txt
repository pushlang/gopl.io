== Нумерация строк ==
:set number					абсолютная нумерация
:set nu
:set nonumber				отключить абсолютную нумерацию
:set nu
:set number!				переключить нумерацию
:set nu!
:set relativenumber			относительная нумерация
:set norelativenumber		отключить относительную нумерацию
:set nornu

== Настройка отображения текста ==
:syntax on					включить подсветку
:syntax off					выключить подсветку (по умолчанию)
:set wrap					разрешить перенос строк
:set nowrap					запретить word wrap

:set tabstop=#				для табуляции используется # пробелов
:set shiftwidth=#			в командах отступа используется # пробелов
:set [no]expandtab			заменять табуляцию на пробелы

== Настройка сворачивания ==
:set foldenable				включить свoрачивание
:set foldmethod=syntax		сворачивание на основе синтаксиса
:set foldmethod=indent		сворачивание на основе отступов
:set foldmethod=manual		выделяем участок с помощью v и говорим zf
:set foldmethod=marker		сворачивание на основе маркеров в тексте
:set foldmarker=bigin,end	задаем маркеры начала и конца блока

== Настройка поиска ==
:set ic	ignorecase			игнорирование регистра при поиске
:set noic
:set is	incsearch			отображение частичных совпадений при поиске
:set nois
:set hls hlsearch			подсветка всех совпадений при поиске
:set noic

== Настройка оформления ==
:color <name>				выбор цветовой схемы ~/.vim/colors/*.vim

:abcd, <Ctrl>+d, <Tab>		дополнить команду, начинающуюся с abcd

== Основы ==
Перемещение
hjkl						перемещение в разные стороны

u, :u[ndo]					отмена предыдущего действия (undo)
<Ctrl>+R, :red[o]			отмена отмены предыдущего действия (redo)

== Операторы и объекты ==
d							удалить			
y							копировать
p							вставить
c							изменить

w							до конца слова, включая пробел.
e							до конца слова, НЕ включая пробел.
$							до конца строки.
^							до начала строки.

== Работа со строками ==
i							режим вставки
I							в начало строки
a							режим добавления
A							в конец строки
R							режим замены 

o							добавить строку после текущей
O							добавить строку перед текущей
cc							удалить строку и начать редактирование
:<n>d						удалить n+1 строку
:<n>y						скопировать n+1 строку

== Выделение ==
v + hjkl					выделение текста
<Shift>+v					выделить строку
<Ctrl>+v					выделение прямоугольника
d							удалить			
y							копировать
p							вставить

gu							к нижнему регистру
gU							к верхнему регистру

:w ИМЯ_ФАЙЛА **				сохранение выделения


== Работа с буфером ==
dd							вырезать строку
yy							копировать строку
p							вставить ниже строки
P							вставить выше строки

"bdd						вырезать строку и поместить в регистр b
"с2d						вырезать три строки и добавить в регистр c
"ayyс						копировать строку в регистр a

:reg [name1][name2][...]    просмотреть содержимое регистров

== Работа с текстом ==
<Ctrl>+p/n					автоматическое дополнение текста (РР)
<Ctrl>+r,=<expr>			вставить выражение (РР)

== Отступы ==
[#]>			сдвинуть выделенное вправо
[#]<			сдвинуть выделенное влево
[#]>>			сдвинуть строку вправо
[#]<<			сдвинуть строку влево

==== Навигация по тексту ====
<Ctrl>+u/d		Page Up / Page Down
<Ctrl>+y/e              Перемотка вверх/вниз без движения курсора  
%                       перейти к парной скобке
0			к началу строки
:<n>                    перейти на строку #n

:m +1 -перемещение вниз на 1 строку
:m -2 -перемещение вверх на 1 строку 

== Маркеры ==
ma                        установить локальный маркер a
mB                        установить глобальный маркер B
`c                        перейти к локальному маркеру c
`0                        вернуться на позицию при закрытии
:marks                    просмотр маркеров
set viminfo='1000,f1      маркеры пишутся в ~/.viminfo

== Сворачивание ==
zc                        свернуть блок
zo                        развернуть блок
zM                        закрыть все блоки
zR                        открыть все блоки
za                        инвертирование
zf                        см :set foldmethod=manual     

== Поиск и замена ==
/Выражение               поиск выражения в файле
\cВыражение              поиск без учета регистра
n                        следующее совпадение
N                        предыдущее совпадение

:#,#s/было/стало/g	замена всех вхождений между двумя строками
:%s/было/стало		замена первого вхождения во всем файле
:%s/было/стало/g	замена всех вхождений во всем файле
:%s/было/стало/gc   	поиск всех вхождений с подтверждением замены

== Работа с файлами ==
:w [fname]		записать изменения
:wa			сохранить изменения во всех файлах
:q			выйти из редактора
:q!			выйти из редактора, не сохраняя изменения
:e **/filename.c	редактировать файл (с поиском по имени)

:pwd			текущий каталог
:cd [path]		перейти в другой каталог
:!команда		выполнить команду, искать по истории

:r ИМЯ_ФАЙЛА **		вставка из файла
:r !ls			вставка вывода командной строки

== Работа с вкладками (a.k.a табами) ==
:tabnew [fname]         создать таб
:tabs                   вывести список табов
:tabn                   следующий таб
:tabp                   предыдущий таб
<n>gt                   перейти на таб #n
gt                      следующий таб
gT                      предыдущий таб
:tabm +1                переместить таб вперед на одну позицию
:tabm -1                переместить таб назад на одну позицию
:tabm 2                 переместить таб на заданную позицию

== Работа с окнами ==
:split <Ctr+w>+L        горизонтальное разбиение
:vsplit <Ctr+w>+J       вертикальное разбиение
<Ctr+w>+r				переключение горизонтальное/вертикальное
<Ctr+w>+с				закрыть окно
<Ctr+w>+{+,-}			изменение высоты текущего окна
:resize 60, :res +5, :res -5
<Ctr+w>+{<,>}			изменение ширины текущего окна
:vertical resize 80, :vertical resize +5, :vertical resize -5
<Ctr+w>, 10, {+,-,<,>}	предустановка шага изменения размера
<Ctr+w>+=				установить равный размер окон
<Ctr+w>+{h,j,k,l}		перемещение между окнами
